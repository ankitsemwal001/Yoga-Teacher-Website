# 🧘 Yoga Teacher Website

A modern, responsive static website built for a Yoga Instructor.  
This project is developed using **HTML, CSS, and JavaScript** with a clean and modular folder structure.

---

## 🚀 Live Demo

(After enabling GitHub Pages)

👉 https://ankitsemwal001.github.io/Yoga-Teacher-Website/

---

## 📁 Project Structure

```
Yoga-Teacher-Website/
│
├── assets/              # Images and media files
├── css/                 # Stylesheets
│   ├── home.css
│   └── pranayama.css
│
├── js/                  # JavaScript files
│   ├── home.js
│   └── include.js
│
├── pages/               # Individual HTML pages
│   ├── home.html
│   ├── ashtanga.html
│   ├── beginner-yoga.html
│   ├── hatha-yoga.html
│   ├── pranayama.html
│   └── ITC.html
│
├── templates/           # Reusable components
│   ├── header.html
│   └── footer.html
│
└── index.html (if added as main entry)
```

## 📌 How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/ankitsemwal001/Yoga-Teacher-Website.git
   ```

2. Open the project folder.

3. Open `home.html` or `index.html` in your browser.
